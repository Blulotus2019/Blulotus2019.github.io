# VS Code 快捷键

## 常用快捷键

- Ctrl+`：打开集成终端
- `Alt+1`：打开侧边栏
- `Ctrl+Alt+F`：格式化
- `Ctrl+Shift+Y`：删除当前行

## 代码片段

- `C+Tab`：console.log('')
